How to contribute to MPC wiki?

To contribute to the wiki you will have to commit on github (see below). Be sure to check our discord

or you can use BETA test1 our editor

Requirements

  • html knowledge
  • MPC knowledge
  • Github account
  • ability to use git (I would not cover that here)
  • (Optional but preferred) being in our discord

Step 1: fork the directory

you can do that by finding a fork button somewhere

Step 2: clone the directory

you can do that by inputing git clone and the path to your wiki's fork

Step 3: make your changes
